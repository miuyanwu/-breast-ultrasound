# 队友方案对比分析报告

**日期**: 2026-06-13  
**对比对象**: 我方 V9 纯分类管线 vs 队友「分割→分类」联合管线

---

## 一、两队方案概览

| 维度 | 我方 (当前) | 队友 |
|------|-----------|------|
| **分类模型** | V9 DISTILL+SOUP+V2.3 加权 logit 融合 | 同 V9（相同模型、相同权重） |
| **分割模型** | 无（仅原图分类） | 2× MiT-B2 UNet 粗分割 + 1× MiT-B2 UNet ROI 精修 |
| **推理策略** | 原图直接分类 | 0.5×原图V9概率 + 0.5×ROI(margin=1.0)V9概率 |
| **分类阈值** | 0.55 | 0.55 |
| **外部数据** | 仅测试集 | 测试集 + 外部 264 张（不含 normal） |

---

## 二、核心差异：「先分割→再分类」

队友方案在此基础上增加了一套分割管线，从原图中定位病灶 ROI，然后对 ROI 区域再次送入 V9 分类器，最终将两个概率 50:50 平均融合。

```
我方:  原图 → V9分类 → 概率
队友:  原图 → V9分类 → 概率₁ (50%)
            → 分割 → ROI提取 → V9分类 → 概率₂ (50%)
                                    ↓
                              最终概率 = 0.5×P₁ + 0.5×P₂
```

## 七、敏感度消融实验（测试集）

队友在测试集上对不同 ROI margin 做了系统搜索：

| 方案 | AUC | F1 | 说明 |
|------|:---:|:---:|------|
| 纯原图 (基线) | 0.9139 | 0.7833 | 我方当前 |
| ROI only (m=1.0) | 0.9245 | 0.7845 | 仅用ROI |
| 0.5×原图 + 0.5×ROI m=1.0 | **0.9225** | **0.7971** | 队友最佳 |
| 0.3×原图 + 0.7×ROI m=1.0 | 0.9245 | 0.7971 | AUC最高但F1持平 |
| 0.7×原图 + 0.3×ROI m=0.25 | 0.9190 | 0.8088 | F1最高但AUC略低 |

结论: 0.5:0.5 + margin=1.0 在 AUC/F1 之间取得最佳平衡。

## 八、关键发现

### 1. 分割→分类是有效的泛化提升策略
- 测试集 AUC +0.86%，外部数据 AUC +1.23%
- **域外泛化收益 > 域内收益**，说明 ROI 聚焦减轻了领域偏移

### 2. 最大收益在敏感度（Recall）
- 测试集敏感度 +1.91%，外部敏感度 +4.44%
- 漏诊（FN）在测试集减少 4 例（51→47），外部减少 4 例（18→14）
- 这对临床筛查场景非常有价值

### 3. 分割质量≠分类收益的线性关系
- 外部数据分割 Dice 仅 0.48（远差于测试集 0.76）
- 但分类 AUC 提升反而是外部的更大（+1.23% vs +0.86%）
- **粗糙的 ROI 定位就已经足够帮助分类器聚焦**

### 4. 分类模型权重完全复用
- 队友未重新训练分类模型，直接使用我方 V9 的三个权重
- 提升完全来自推理阶段的 pipeline 改进，不需要额外训练成本

---

## 九、建议

1. **立即采纳分割→分类管线**：推理阶段改进，无需重新训练，AUC/F1/Sensitivity 全面正向
2. **ROI margin 可固定为 1.0**：消融实验已验证这是 AUC/F1 最佳平衡点
3. **分割模型可进一步优化**：外部数据 Dice 仅 0.48，如果用更强的分割架构或域适应，分类还有进一步提升空间
4. **考虑将分割管线整合进 Web 应用**：单病例诊断 + 高通量筛查均可受益于更准确的概率估计
5. **外部数据上的弱特异性值得关注**：队友特异度略降（88.51%→87.93%），但敏感度大幅提升远超补偿

---

## 十、总结

| 维度 | 评价 |
|------|------|
| 分类核心 | 不变，双方共用 V9 |
| 增量贡献 | 分割管线 + ROI 联合推理 |
| 测试集收益 | AUC +0.86%, F1 +1.38%, Sens +1.91% |
| 外部泛化 | AUC +1.23%, F1 +2.16%, Sens +4.44% |
| 训练成本 | 零（复用现有分类权重） |
| 推理成本 | 额外跑一次分割 + 一次 ROI 分类 |
| **综合结论** | ✅ 值得采纳，建议整合进主线 |
